# Displayer

An imaging tool aimed to visualize location data contained in images.


## Usage
```
$ cd displayer
$ cargo run -- --port <PORT> --data <DIRECTORY_PATH>
```
Meaning of flags: <br/>
-> hosting on http://localhost:PORT <br/>
-> recursive search for images in DIRECTORY_PATH
